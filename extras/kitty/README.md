# Mono Glow for Kitty

Part of [monoglow.nvim](https://github.com/wnkz/monoglow.nvim).

## Installation

1. Copy the `.conf` files to your Kitty config directory:
   - **macOS/Linux:** `~/.config/kitty/`
2. Add to your `kitty.conf`:
   ```
   include monoglow_z.conf
   ```
3. Available themes: `monoglow_z.conf`, `monoglow_lack.conf`, `monoglow_void.conf`
