# Mono Glow for Helix

Part of [monoglow.nvim](https://github.com/wnkz/monoglow.nvim).

## Installation

1. Copy the `.toml` files to your Helix themes directory:
   - **macOS/Linux:** `~/.config/helix/themes/`
   - **Windows:** `%AppData%\helix\themes\`
2. Add to your `config.toml`:
   ```toml
   theme = "monoglow_z"
   ```
3. Available themes: `monoglow_z`, `monoglow_lack`, `monoglow_void`
