# Mono Glow for VS Code

A dark, mostly monochrome colorscheme with a touch of glowing.

Part of [monoglow.nvim](https://github.com/wnkz/monoglow.nvim).

## Local Installation

1. Copy or symlink this folder to your VS Code extensions directory:
   - **macOS/Linux:** `~/.vscode/extensions/monoglow`
   - **Windows:** `%USERPROFILE%\.vscode\extensions\monoglow`
2. Restart VS Code
3. Open Command Palette (`Cmd+Shift+P` / `Ctrl+Shift+P`) and select `Preferences: Color Theme`
4. Select **Mono Glow**, **Mono Glow Lack**, or **Mono Glow Void**
