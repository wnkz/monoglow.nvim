# Mono Glow for Vim

Part of [monoglow.nvim](https://github.com/wnkz/monoglow.nvim).

## Installation

1. Copy the `.vim` files to your Vim colors directory:
   - **Unix:** `~/.vim/colors/`
   - **Windows:** `$HOME\vimfiles\colors\`
2. Add to your `.vimrc`:
   ```vim
   colorscheme monoglow-z
   ```
3. Available colorschemes: `monoglow-z`, `monoglow-lack`, `monoglow-void`
