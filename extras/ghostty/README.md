# Mono Glow for Ghostty

Part of [monoglow.nvim](https://github.com/wnkz/monoglow.nvim).

## Installation

1. Copy the theme files to your Ghostty themes directory:
   - **macOS/Linux:** `~/.config/ghostty/themes/`
2. Add to your `~/.config/ghostty/config`:
   ```
   theme = monoglow_z
   ```
3. Available themes: `monoglow_z`, `monoglow_lack`, `monoglow_void`
